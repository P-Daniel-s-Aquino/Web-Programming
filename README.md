## 📁 Proyecto De Programación Web I

### TP-IndieBAires
**Plataforma de venta de tickets para recitales en Buenos Aires**

Una aplicación web que permite a los usuarios explorar y comprar entradas para eventos musicales en la ciudad de Buenos Aires. 

#### Características principales:
- 🎵 **Catálogo de bandas**: Listado dinámico de artistas con información de recitales
- 📅 **Detalles de eventos**: Fecha, horario, lugar y precio de cada presentación
- 🛒 **Sistema de compra**: Carrito de compras y generación de tickets
- 👥 **Página "Nosotros"**: Historia y misión de la plataforma
- 📝 **Sección de comentarios**: Espacio para que usuarios compartan recomendaciones

#### Tecnologías utilizadas:
- HTML5
- CSS3
- Diseño responsive

#### Páginas:
- `IndieBAires.html` - Página principal con catálogo de eventos
- `compra.html` - Simulación de ticket de compra
- `nosotros.html` - Información sobre la plataforma

#### Cómo acceder:
1. Abre `IndieBAires.html` en tu navegador
2. Explora el catálogo de bandas
3. Haz clic en "Comprar" para ver el detalle de la compra
4. Visita la sección "Nosotros" para conocer más sobre IndieBAires
